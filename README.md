# 🔴 Women Safety Analytics — India

## 📌 About This Project
This project analyzes crimes against women 
across Indian states using real government 
data. Built using Python as part of my 
Data Analytics learning journey.

## 🛠️ Tools & Technologies Used
- Python 🐍
- Pandas
- Matplotlib
- CSV Dataset

## 📊 What This Project Shows
- Top 10 Indian states with highest crimes 
  against women
- Real data visualization using bar charts

## 🔍 Key Findings
- Uttar Pradesh has the highest crime rate
- West Bengal and Madhya Pradesh follow
- Southern states are comparatively safer

## 📁 Files in This Project
- analysis.py — Main Python code
- CrimesOnWomenData.csv — Dataset
- top_states.png — Generated chart
